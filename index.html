Cats & Dogs & Co

Cats & Dogs & Co (in further text CDC) is service intended for tracking your pets.

Contrary to all other similar projects where you buy "tracking band" this is do-it-yourself project. We provide two Web Services (Soap JAX-WS) where you can get your data (to use in your application) or send your data from "Tracker".

Tracking Software

Most of companies allow you to buy their tracker and then you can follow it's location on their web site. Our service is not like that... We actually don't have display on Web, but you can use other means to get and see your data. (We might create simple Web site in the future where there will be possible to add user, animals and see location, but this is not planned at the moment) 1. Making your own application and using supplied Web Services 2. Use one of our application: 2a) .NET application. This simple Windows application which can be used until our "official" application comes out. If you are Windows user you can try this out, or else wait a little longer and 2b will be out. COMMING in November 2015. 2b) Java application. Application for displaying location data, this is actually official application for this service. Client is open source and can be extended by users (running on Sourceforge/GIT). COMMING IN 1st Quarter of 2016.

Tracker

Like I said most companies have product which can be used for tracking your animal... we don't. Idea of whole project was to either build your own tracker, which can then send data to our service, or use old phone to do that for you.

Use your old (Android) phone

Now this is probably the cheapest sollution. Use your old Phone, or buy some cheap one (you can get cheap Android 4 phone for under 50 EUR). Phone needs to have GPS though. Most of old Android 4.x phones have that, but make sure before you get. Now you need "data SIM card" of one of your Mobile Phone Service Providers (you can get that cheap too in most countries and our android application. COMMING in November 2015.

Use your old phone (not Android)

For no other phones are supported, but since this is OSS you can build your own application and share it with others. What you need to do:

  • retrieve Animal Id for specific Username and AnimalName
  • create 60 seconds loop:
    + read GPS location + write location to CatsAndDogsAndCoTransmiter Web Service (time is in long format - create String and parse it into long (for example: 20. April 2015 14:01:04 would be 20150420140104 - year(4)month(2)day(2)hour(2 - 24h format)minute(2)second(2))

If you decide to do this please share your code in our repository on git under (transmiter//). Trasnmiter name in case we get more implementation for specific type (Android, iPhone, etc)

Build your own transmitter

If you are good with your hands you can try to build your own transmitter. There might be several options for you, but easiest and cheapest would probably be to use Arduino.
